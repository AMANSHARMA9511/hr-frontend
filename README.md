# 🎨 Frontend - Employee Leave & Attendance Management System

## 📌 Overview
This frontend application provides the user interface for the Employee Leave & Attendance Management System. It allows employees and admins to interact with the system efficiently through a clean and responsive UI.

---

## 🛠 Tech Stack

- Next.js (React Framework)
- Tailwind CSS
- Axios (API calls)
- React Hot Toast (Notifications)

---

## 📂 Project Structure
frontend/
│── pages/
│── components/
│── services/
│── styles/
│── public/


---

## 🚀 Features

### 👨‍💼 Employee Features
- Register & Login
- Apply for leave
- View leave history
- Mark daily attendance
- View attendance records
- View remaining leave balance

### 👨‍💻 Admin Features
- Login as Admin
- View all employees
- Approve/Reject leave requests
- Monitor attendance records

---

## 🔐 Authentication & Authorization
- JWT-based authentication
- Role-based UI rendering (Admin / Employee)
- Protected routes

---

## 🎯 UI Highlights
- Clean and responsive design
- Mobile-friendly layout
- Toast notifications for success/error
- Form validation
